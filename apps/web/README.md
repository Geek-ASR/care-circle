# apps/web

The CareCircle frontend — a Vite + React + TypeScript single-page app.

This package is part of an npm workspaces monorepo; run everything from
the **repository root** (`npm install`, `npm run dev`, etc. — see the
[root README](../../README.md)) rather than from inside this folder.

For architecture, database schema, deployment, and environment variable
docs, see [`../../docs/`](../../docs/).
